# MdConvertor

تبدیل مارک‌داون به PDF با پشتیبانی کامل از **راست‌چین** و **چپ‌چین** و متن‌های **ترکیبی** (فارسی + انگلیسی)

## نصب

```bash
npm install
```

## استفاده

```bash
node index.js input.md output.pdf [options]
```

## مثال‌ها

```bash
# ساده
node index.js sample.md output.pdf

# فارسی با کاور و فهرست
node index.js sample.md output.pdf --lang fa --title "مستندات پروژه" --author "علی رضایی" --cover --toc

# قالب تیره
node index.js sample.md output.pdf --theme dark --lang fa

# انگلیسی با سایز کاغذ دلخواه
node index.js sample.md output-en.pdf --lang en --page-size Letter

# لندسکیپ با شماره صفحه
node index.js sample.md output.pdf --orientation landscape --page-numbers
```

## راهنما

### گزینه‌های اصلی

| گزینه       | پیش‌فرض  | توضیحات                        |
|-------------|----------|----------------------------------|
| `--lang`    | `en`     | کد زبان (`fa`, `ar`, `en`, ...) |
| `--rtl`     | —        | اجبار راست‌چین                  |
| `--ltr`     | —        | اجبار چپ‌چین                    |

### ظاهر و صفحه‌بندی

| گزینه             | پیش‌فرض     | توضیحات                                    |
|-------------------|-------------|----------------------------------------------|
| `--theme`         | `light`     | قالب رنگی: `light`, `dark`, `sepia`          |
| `--page-size`     | `A4`        | اندازه صفحه: `A4`, `Letter`, `Legal`, `A3`, `A5` |
| `--orientation`   | `portrait`  | جهت صفحه: `portrait`, `landscape`            |
| `--margin`        | `20`        | حاشیه صفحه (mm)                              |
| `--font-size`     | `11`        | اندازه قلم (pt)                              |
| `--line-height`   | `1.9`       | فاصله خطوط                                   |

### محتوا

| گزینه           | پیش‌فرض  | توضیحات                            |
|-----------------|----------|--------------------------------------|
| `--title`       | —        | عنوان سند (برای کاور)               |
| `--author`      | —        | نویسنده (برای کاور)                 |
| `--cover`       | —        | افزودن صفحه کاور                    |
| `--toc`         | —        | افزودن فهرست مطالب                  |
| `--page-numbers`| —        | نمایش شماره صفحات در فوتر           |
| `--no-google-fonts` | —    | عدم لود فونت از گوگل (آفلاین)      |

## ویژگی‌ها

- **RTL/LTR خودکار** — تشخیص جهت متن در سطح هر پاراگراف
- **متن‌های ترکیبی** — پشتیبانی از مخلوط فارسی و انگلیسی در یک پاراگراف
- **سه قالب رنگی** — روشن (`light`)، تیره (`dark`) و سپیا (`sepia`)
- **کاور اختصاصی** — صفحه جلد با عنوان، نویسنده و تاریخ
- **فهرست مطالب** — استخراج خودکار هدینگ‌ها
- **صفحه‌آرایی حرفه‌ای** — border-radius، سایه، گرادینت، فونت مناسب
